What are IDE's

A IDE is a software application that helps programmers develop software code efficiently in increases developer productivity by combining capabilities such as software editing building testing packaging in an easy to use application

Visual studio is a great IDE it is a lightning fast code editor which supports many programming languages it has syntax higlighting lots of key shortcuts

PyCharm:

PyCharm is another great IDE like Visual Studio Code also it supports many different programming languages such as JavaScript Python and many more so it is a great platform to write python code
